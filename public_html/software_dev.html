How it All Started
.......................................

My programming journey began in Grade 8. My first project was an Operating System in the C language. Trust me, this project hasn't been completed till date :D I tried out open-source Operating Systems such as KolibriOS, Minix, Menuet and so on.
Filled with energy, curiosity, and an insatiable thirst for knowledge, I kept expantiating my knowledge in not just programming, but also computer architectures, memory mapping and configuration, and so much more. Being called a programmer isn't enough, you have to find a practice, say Game Development or Software Development in order to channel your programming knowledge and energy.
I started out with the C programming language, and later switched to x86 Assembly Language (Yes, you guessed it. I was crazy right from the start). Whenever I am learning a new topic, I always start with the intention to move to the top.
